# AI Baseball Game 🏟️

세 자리 숫자야구 게임 - AI가 사용자가 입력한 숫자를 맞추는 게임

## 📋 개요

사용자는 투수 역할을 하며 세 자리 숫자를 입력하고, AI 타자가 그 숫자를 맞추려고 시도합니다. AI는 매번의 추측에 대한 피드백(스트라이크/볼)을 활용하여 점진적으로 정답에 접근합니다.

## 🎮 게임 규칙

### 숫자 규칙
- 세 자리 숫자 (예: 123, 456, 789)
- 1-9 사이의 숫자만 사용 가능
- 중복된 숫자는 사용 불가

### 스트라이크와 볼
- **스트라이크**: 정확한 자리에 정확한 숫자
- **볼**: 다른 자리에 숫자가 존재
- **3스트라이크**: AI가 정답을 맞춘 것!

### 예시
```
정답: 123

AI 추측: 132
→ 1스트라이크 2볼 (1은 정확한 자리, 2와 3은 있지만 자리 틀림)

AI 추측: 123
→ 3스트라이크 (성공!)
```

## 🎯 게임 플레이

1. 게임을 시작하면 세 자리 숫자를 입력하라는 메시지가 나타납니다.
2. 1-9 사이의 숫자로 중복 없이 세 자리 숫자를 입력합니다.
3. AI가 계속해서 숫자를 추측합니다.
4. 각 추측마다 스트라이크와 볼이 표시됩니다.
5. AI가 3스트라이크를 얻으면 성공입니다!
6. 여러 라운드를 플레이할 수 있으며, 통계가 기록됩니다.

## 🤖 AI 알고리즘

AI는 다음과 같은 전략을 사용합니다:

1. **첫 추측**: 완전 랜덤으로 세 자리 숫자 선택
2. **이후 추측**: 
   - 이전 추측과 피드백을 분석
   - 모든 가능한 조합 중에서 이전 피드백과 일치하는 것만 필터링
   - 필터링된 조합 중에서 랜덤 선택

이 방식으로 AI는 피드백을 활용하여 효율적으로 정답에 접근합니다.

## 📊 통계

게임 종료 시 다음 통계가 표시됩니다:
- 총 플레이한 라운드 수
- 평균 시도 횟수
- 최고 기록 (가장 적은 시도 횟수)
- 최악 기록 (가장 많은 시도 횟수)

## 🔧 코드 구조

### 주요 클래스
